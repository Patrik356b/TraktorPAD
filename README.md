# TraktorPAD
Traktor mapping for the Novation Launchpad (Based on Mini)

WIP

Top row buttons are:
- File browser UP
- File browser DOWN
- Load Selected track into Deck A
- Load Selected track into Deck B

Also; basic VU meter effect happening on the round buttons (top and right side)
